# Workout Tracker

A full-stack workout tracking app for logging sessions, tracking exercise performance, and reviewing progress with charts.

## Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: SQLite
- Charts: Chart.js

## Features
- Create workout sessions
- Add multiple exercises and sets
- Save weight, reps, and notes
- View workout history
- Track exercise progress over time
- Visualize top weight and training volume

## Run locally

```bash
cd workout-tracker
npm install
npm start
```

Then open:

```bash
http://127.0.0.1:3000
```

## Pages
- `/` dashboard
- `/workout.html` new workout
- `/history.html` workout history
- `/progress.html` exercise progress
